## 📖 About The Project

Aurelia is a modern, beautifully designed web application for a fictional women's accessories brand. The website showcases a full e-commerce front-end experience where users can browse various collections of jewelry, bags, and luxury accessories.

## 🛍️ Categories

The store features a wide range of curated categories:
- **Necklaces & Jewelry:** Delicate pieces ranging from everyday wear to statement pieces.
- **Bags & Clutches:** Discover stylish essentials for any occasion.
- **Rings:** Beautifully crafted rings perfect for stacking or wearing alone.
- **Bracelets:** Elegant bracelets designed to complement any outfit.
- **Earrings:** Handcrafted earrings ranging from studs to elegant drops.
- **Scarves:** Premium quality scarves to complete your look.

## 🌟 Key Features & Pages

- **Home Page (`/`):** A stunning landing page featuring hero banners, new collections, category quick-links, and featured products. 
- **Product Details (`/product/:id`):** Deep dive into individual items with high-quality images, descriptions, pricing, and sizing/material information.
- **Collections (`/collections`):** A centralized page to browse all the curated selections.
- **About Us (`/about`):** Shares the brand story, values, craftsmanship, and the inspiration behind the products.
- **Account & Profile (`/account`, `/profile`):** User dashboard for managing personal details and order history.
- **User Authentication (Required Actions):** 
  - To utilize the **Add to Bag** feature, users must be logged in.
  - Using the **Wishlist** (saving favorite items for later checkout) also strictly requires an active user session/login. 
- **Checkout (`/checkout`):** A streamlined, mockup checkout process for reviewing your cart and simulating a purchase flow.

## 🚀 Getting Started Locally

If you'd like to run this demo project locally:

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`
4. Open the local link (usually `http://localhost:8080`) in your browser.
